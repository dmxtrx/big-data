我的“大数据技术”课程作业，主要使用 **PySpark** 做分布式计算和流式处理。

**技术栈：**`PySpark`（RDD/DataFrame）、`Kafka`、`Spark Streaming`、`MLlib`。

* **[作业 1：RDD 基础](BigData_HW1.md)**  
  练习 RDD 变换，计算莫斯科地点距离，并实现电影余弦相似度。

* **[作业 2：DataFrame 与可视化](BigData_HW2.md)**  
  分析 NYC CitiBike 数据，关联地理信息，画出出行热力图和区域分布图。

* **[作业 3：实时流处理](BigData_HW3.md)**  
  搭建 **Telegram → Kafka → Spark** 流水线，做滑动窗口统计和简单专有名词统计。

* **[作业 4：推荐系统 (MLlib)](BigData_HW4.md)**  
  使用 **ALS** 训练 MovieLens 推荐模型，并和基础协同过滤方法做对比。
